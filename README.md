# Sales-Forecasting-System
📈 Sales Forecasting System using Linear Regression
📌 Project Overview

This project is a simple Sales Forecasting System built using Python and Machine Learning. It uses Linear Regression to predict future sales based on historical sales data. The dataset is synthetically generated to simulate real-world increasing sales trends.

🚀 Features
Generates synthetic sales dataset for 60 days
Applies Linear Regression for prediction
Forecasts future sales for next 10 days
Visualizes:
Actual sales data
Trend line
Future predictions
Displays predicted values in console output
🛠️ Technologies Used
Python 🐍
Pandas
NumPy
Matplotlib
Scikit-learn
